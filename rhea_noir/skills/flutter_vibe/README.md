## Usage

```python
from rhea_noir.skills.flutter_vibe import skill

# Generate app scaffold
result = skill.execute("scaffold_app", name="vibe_app", description="Premium dashboard")

# Generate widget
result = skill.execute("generate_widget", prompt="Glassmorphic card with blur", name="GlassCard")

# Review code
result = skill.execute("review_code", code="<your dart code>")
```
