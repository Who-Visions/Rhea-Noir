# Brand Identity Guidelines

## Colors
- **Primary**: #000000 (Black)
- **Secondary**: #FFFFFF (White)
- **Accent**: #FF5733 (Orange)
- **Background**: #F5F5F5 (Light Gray)

## Typography
- **Headings**: Inter, Bold
- **Body**: Roboto, Regular
- **Monospace**: Fira Code (for code blocks)

## UI Elements
- **Border Radius**: 8px
- **Shadows**: Soft, diffuse (e.g., `0 4px 6px -1px rgba(0, 0, 0, 0.1)`)
- **Spacing**: 8pt grid system

- **Main**: `assets/logo-full.svg`
